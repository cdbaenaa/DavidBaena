Project 3

Rockbuster Stealth Data Analysis Project

Case Study

Project Overview

Rockbuster Stealth LLC is a fictitious movie rental company that used to have stores worldwide. Facing stiff competition from streaming services such as Netflix and Amazon Prime, the Rockbuster Stealth management team plans to use its existing movie licenses to launch an online video rental service to stay competitive.

As an Analyst of the business intelligence (BI) department, the goal is to help with the launch strategy for the new online video service. In addition, the BI department helps other departments with data-related queries, from inventory to customer insights. The first task is to load all of Rockbuster’s data into a relational database management system (RDBMS). And then use SQL to analyse the data and answer any ad-hoc business questions other departments may have.

The main questions they’d like to answer:

● Which movies contributed the most/least to revenue gain?
● What was the average rental duration for all videos?
● Which countries are Rockbuster customers based in?
● Where are customers with a high lifetime value based?
● Do sales figures vary between geographic regions?
